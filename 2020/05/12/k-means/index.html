K-prototypes

K-prototypes = K-means (numerical) + K-modes (categorical)

cost function: $$E=\sum^K_{l=1}\sum^n_{i=1}y_{il}\ \text{d}(X_i,Q_l)$$

$X_i=\{x_{i1},x_{i2},\cdots,x_{im}\},\ Q_l=\{q_{l1},q_{l2},\cdots,q_{lm}\}$

$y_{il}\in \{0,1\}$，1 表示样例 $X_i$ 分在 $l$ 簇。$\displaystyle \sum^K_{l=1}y_{il}=1 \ (\forall i=1,\cdots,n).$

把样例分派给第 $l$ 簇的损失：

$$E_l=\sum^n_{i=1}y_{il}\ \text{d}(X_i,Q_l)$$

当 $\displaystyle q_{lj}=\frac{1}{n_l}\sum^n_{i=1}y_{il}\ x_{ij}$ 时，$E_l$ 最小。

$$\begin{aligned}
\text{d}(X_i,Q_l)&=\sum^{m_r}_{j=1}(x^r_{ij}-q^r_{lj})^2+\gamma_l\sum^{m_c}_{j=1}\delta(x^c_{ij},q^c_{lj})\\\
&=E^r_l+E^c_l\end{aligned}$$

$$\delta(x,q)=\left\{\begin{array}{l}
1, \ x\neq q\\\
0, \ x=q
\end{array}\right.$$

$E^r_l$ 是 numerical variables 的损失，$E^c_l$ 是 categorical variables 的损失。

$\gamma_l$ 的取值，一般是根据所在簇的 numerical variables 的标准差 $\sigma$来定，$\gamma\in [1/3\sigma,2/3\sigma]$。
